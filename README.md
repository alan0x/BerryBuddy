# 🍓 BerryBuddy

A tiny AI voice companion built on Raspberry Pi Zero 2 W.  
Made with love for my child ❤️

---

## 🌟 What is BerryBuddy?

BerryBuddy is a small, friendly AI that you can talk to.

Press a button, speak, and it will listen, think, and reply with a voice — just like a little companion living inside a Raspberry Pi.

---

## 🎯 Features

- 🎤 Voice input (microphone)
- 🧠 AI conversation powered by Ark API
- 🔊 Voice responses (text-to-speech)
- ⚡ Lightweight — runs on Raspberry Pi Zero 2 W
- ❤️ Designed as a fun and educational companion for kids

---

## 🧩 Hardware

- Raspberry Pi Zero 2 W
- Micro SD card (16GB+)
- USB microphone (or I2S mic)
- Speaker / headphone output
- Optional: button for push-to-talk
